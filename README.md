# homebrew-arqtos

Homebrew tap for the [arqtos](https://github.com/arqtiqa/arqtos) toolkit.

## Install (when a release lands)

```bash
brew tap arqtiqa/arqtos
brew install arqtos
```

## Status

**Pre-1.0** — name claim only; no formula yet. The first formula lands when arqtos hits a tagged stable release. Until then, install from source per the toolkit's [CONTRIBUTING.md](https://github.com/arqtiqa/arqtos/blob/main/CONTRIBUTING.md#quick-start).
